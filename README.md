# 🌐 NGO Web Platform

This project is a modern web application built for a Non-Governmental Organization (NGO).  
The goal is to create a platform where the organization can manage its content, projects, and volunteers, while also presenting itself professionally online.

---

## 💡 About the Project

This platform includes both:
- a **public website** for visitors
- an **admin dashboard** for managing everything

Users can explore the NGO, see projects, read news, and even apply as volunteers.  
Admins can easily manage content like news, reports, media, staff, and projects.

The project is built using the **MERN Stack** (MongoDB, Express, React, Node.js).

---

## 🚀 Main Features

### For Users
- View information about the NGO (About, Mission, Vision)
- Explore projects and events
- Read news and reports
- View media (photos & videos)
- See staff/team members
- Apply as a volunteer
- Use an AI chatbot for help
- Switch language (Albanian 🇦🇱 / English 🇬🇧)
- Works on mobile and desktop

---

### For Admin
- Secure login system
- Manage:
  - News & announcements
  - Reports (PDFs)
  - Media (images/videos)
  - Projects & events
  - Staff members
- View and manage volunteer applications
- Dashboard with basic statistics
- Add content in both languages

---

## 🤖 AI Chatbot

The platform includes a simple AI chatbot that helps users:
- Get information about the NGO
- Learn about projects
- Find how to apply as a volunteer  

It works in both Albanian and English.

---

## 🌐 Multilingual Support

- The website supports **2 languages**
- Users can switch language with a button
- Content changes dynamically
- Language preference is saved in the browser
